# KJV Reese Chronological Study Bible

An interactive web application for reading the **King James Bible** arranged in chronological order, following the editorial framework of the *KJV Reese Chronological Study Bible* (Edward Reese, Baker Publishing Group, 2016).

## Features

- 📖 **Chronological Reading** -- Scripture arranged from Eternity Past through the Writings of John
- 🔍 **Search** -- Filter sections and eras by keyword
- 🔖 **Bookmarks** -- Save and return to any section
- ✅ **Reading Progress** -- Mark sections complete and track your progress
- 📝 **Personal Notes** -- Write study notes per section
- 🔠 **Adjustable Font Size** -- Comfortable reading at any size
- 🌑 **Dark Vellum Theme** -- Elegant illuminated-manuscript aesthetic

## Tech Stack

- Framework: React 18 + Vite
- Icons: Lucide React
- Fonts: Cinzel, EB Garamond, Crimson Pro (Google Fonts)
- Bible API: [bible-api.com](https://bible-api.com) (free, KJV)
- Storage: `localStorage` (bookmarks, progress, notes)

## Getting Started

```bash
npm install
npm run dev
```
